# Formula 1 Race Prediction Using MATLAB

## Overview

This project analyzes historical Formula 1 race data and builds a basic predictive model using MATLAB. It processes driver, team, and race performance data to identify trends and estimate race outcomes.

## Features

* Data cleaning and validation of race datasets
* Driver performance error checking and consistency analysis
* Team and driver points comparison
* Visualization of race and team performance using bar graphs
* Basic predictive logic based on historical performance trends and user input

## Files Included

* `Mainscript_Ibishomi.m` – Main execution script for the project
* `Loop2.m`, `Loop3.m` – Data processing loops for analysis
* `Errorcheck_Driver.m` – Validates driver data consistency
* `Bargraph_Colours.m` and variants – Visualizations of race and team data
* `NewRaceResults.xlsx` – Historical race results dataset
* `NewTeamPoints.xlsx` – Team championship points dataset
* `NewDriverSheet.xlsx` – Driver performance dataset

## Tools Used

* MATLAB
* Excel for dataset organization
* Data visualization (bar charts and comparative analysis)

## Purpose

The goal of this project was to apply MATLAB programming to real-world inspired motorsport data, focusing on data analysis, visualization, and basic predictive modeling.
